# Ping-Pong 
A ping-pong ball game in Python using Turtle module

##### Requirements : Turtle module

#### How to Play
If a player misses to hit the ball, he loses a point to other player

Player 1:

W- Move Up
S- Move Down

Player 2:        
&#8593; - Move Up
&#8595; - Move Down

### Clone it, Run the file and play!
